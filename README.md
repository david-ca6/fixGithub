# FixGitHub

Chrome extension that fixes GitHub by restoring the original Octocat to the top left and adding kittens where needed. Because kittens make everything better. [AI SLOP]

> This chrome extension is AI slop because I have no time to waste on GitHub... and hate JS/TS... but though it would be funny to have cats on GitHub. So here we are.

## Features

- Replaces the top-left GitHub `modern silhouette Octocat` logo with a `classic Octocat`.
- Replaces Copilot logos with a `cyber kitten`.
- Adds ` ~nya` to GitHub user names.

## Install locally

1. Open `chrome://extensions`.
2. Enable Developer mode.
3. Click Load unpacked.
4. Select this folder.
5. Visit `https://github.com`.
